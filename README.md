# ATM 10-Finger Security Analysis 🔐

## Problem Statement
Analyzed sample ATM transaction data. Key finding: 100% transactions during night hours had "Failed" status. This indicates high security risk for banks.

## My Solution Approach
1. Designed 10-Finger biometric verification app UI for enhanced security
2. Used Google Sheets to analyze transaction patterns with COUNTIF & Charts
3. Identified Risk Pattern: Night Time = 100% Failure Rate
4. Location Insight: Hyderabad showed highest transaction count

## Business Recommendation 
Implement extra OTP/3-Finger verification for night-time transactions & high-risk locations. This reduces fraud risk by adding a security layer.
